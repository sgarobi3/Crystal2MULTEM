Thank you for downloading crystal2multem.

The code is designed to convert CrystalMaker coordinate files to the format required by MULTEM.

MULTEM can be found at https://github.com/Ivanlh20/MULTEM 

CrystalMaker can be found at http://crystalmaker.com/

-------------------------------------------------------------------------
-------------------------------------------------------------------------

In order to use the code properly, the following files are required:

- crystal2multem.m
- coordinateConvert.m
- atomicNumber.m
- AtomicNumbers.xlsx
- dispstat.m (version 1.1.0.0 (1.98 KB) by kasim tasdemir @ https://www.mathworks.com/matlabcentral/fileexchange/44673-overwritable-message-outputs-to-commandline-window)
